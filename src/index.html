Условия и Требования
Условия на своем авто:
  • Подключаем к системе UBER под 10%.
  • Ни каких взносов 0%
  • Можем подключать удаленно без приезда в офис
  • ИП не требуется
  • Заказы распределяются без учета рейтинга ближайшим водителям
  • Работа только по безналичному расчету
  • Деньги перечисляем на карту водителю или наличными в офисе
Условия на авто компании:
  • Предоставляем авто без залога
  • Без ежедневных выплат по аренде
  • Cреда пятница и суббота работа на компанию остальные дни вы работаете на себя
Требования:
  • Наличие автомобиля без рекламы
  • Машина не старше 3 лет
  • Автомобиль не должен быть желтый и не с желтыми номерами
  • Стаж вождения не менее 3 лет
  • Девайс на системе андроид 4.0 и выше(самсунг)
Документы:
  • Паспорт лицевая сторона
  • Права 2 стороны
  • СтС
  • Лицензия(если есть)
  • номер телефона
  • Индефикационый код
  • Номер карты для перечисления денег
